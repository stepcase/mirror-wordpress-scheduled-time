This plugin displays not only the scheduled Data, but also the scheduled Time of a WordPress Post.

== Description ==

When you're having a large blog and post mulitple blogposts per day with multiple authors it can be very interesting to see the actual time a post is scheduled.
Standard WordPress behaviour is that you only see the scheduled date when a post is scheduled.
This plugin's main feature is that is displays also the WordPress Scheduled Post TIME. 

Other features:

* Sorable Column on Post Status
* Gives a clear overview if a post is published / scheduled / draft / pending review or Trashed with different colors.

== Installation ==

1. Upload the directory 'wordpress-scheduled-time' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress

OR 

Install the plugin via the WordPress Plugin Repository
